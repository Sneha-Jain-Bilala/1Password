# Brand Logos

Drop official logo image files in this folder using lowercase filenames.

Current resolver mapping in `lib/backend/service_logo_resolver.dart` expects:

- gmail.png
- google.png
- google_maps.png
- netflix.png
- github.png
- amazon.png
- twitter.png
- x.png
- facebook.png
- instagram.png
- linkedin.png
- spotify.png
- apple.png
- microsoft.png
- discord.png
- dropbox.png
- paypal.png
- rupay.png
- visa.png
- mastercard.png
- amex.png
- youtube.png
- whatsapp.png
- telegram.png

Notes:
- Prefer transparent PNGs with square canvas (e.g. 128x128).
- Use official brand assets from each product's brand guidelines.
- If a file is missing, the app falls back automatically to the icon-pack logo.
